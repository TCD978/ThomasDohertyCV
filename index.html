Problem‑Solving Case Study: Automated GitHub‑Powered Project Feed

The Problem

My digital CV originally required manual updates every time I created, renamed, or improved a project. This slowed iteration, created inconsistencies, and made the CV feel static despite ongoing work.

The Approach

Built a modular system that discovers repositories via the GitHub API, selects and sorts projects by recency and relevance, and lists their names in a compact Quick Links panel on my CV; each link opens the repository on GitHub.

The Solution

  • Fetched repository data from the GitHub API.
  • Filtered out forks and non‑portfolio repositories.
  • Sorted projects by recency and relevance.
  • Injected project names into the Software Projects list while preserving the design system.
  • Isolated selection logic in a single file for maintainability.

Impact

My CV now updates automatically, requires zero manual maintenance, and always reflects my most recent engineering work.

It demonstrates my ability to identify inefficiencies, design a modular system around them, and deliver a clean, reliable automation solution.

Code Used in This Project (projects.js)



// projects.js — Auto-load GitHub repositories into the CV

document.addEventListener("DOMContentLoaded", () => {
  const list = document.getElementById("projects-list");
  if (!list) return;

  const username = "TCD978";
  const apiURL = `https://api.github.com/users/${username}/repos?sort=updated`;

  async function loadRepos() {
    try {
      const res = await fetch(apiURL);

      if (!res.ok) {
        list.innerHTML = `
  • Unable to load projects.
    • `;
        return;
      }

      const repos = await res.json();

      // Filter out forks
      const filtered = repos.filter(repo => !repo.fork);

      if (filtered.length === 0) {
        list.innerHTML = `
  • No public projects found.
    • `;
        return;
      }

      // Clear loading text
      list.innerHTML = "";

      // Build list items
      filtered.forEach(repo => {
        const li = document.createElement("li");
        li.className = "project-item";

        const link = document.createElement("a");
        link.href = repo.html_url;
        link.target = "_blank";
        link.rel = "noopener";
        link.textContent = repo.name;

        li.appendChild(link);

        // Optional: add description if available
        if (repo.description) {
          const desc = document.createElement("p");
          desc.className = "project-desc";
          desc.textContent = repo.description;
          li.appendChild(desc);
        }

        list.appendChild(li);
      });

    } catch (err) {
      console.error("GitHub fetch error:", err);
      list.innerHTML = `
  • Network error loading projects.
    • `;
    }
  }

  loadRepos();
});
